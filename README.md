# FridayTrader 🤖

An AI-powered paper trading agent built with Claude (Anthropic API) + NemoClaw.

## Vault Structure
- **Trades/** — every trade decision logged with AI reasoning
- **Reports/** — daily/weekly performance summaries
- **Strategy/** — trading rules and strategy notes
- **Agent-Logs/** — raw agent activity logs

## Project Goals
- Build an autonomous paper trading agent
- Document AI reasoning behind every decision
- Track performance over time
- University application project
