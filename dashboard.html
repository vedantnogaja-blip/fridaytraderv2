How FridayTrader works
Every component that powers the autonomous trading agent
🧠
AI Reasoning
Every trade decision is made by Claude (Anthropic API). The agent analyses price momentum, volume, news sentiment and portfolio risk before deciding to buy, sell or hold — and writes its reasoning in plain English.
📰
News Sentiment
Before every trade the agent fetches the latest headlines for each stock and asks Claude to rate the news as Bullish, Bearish or Neutral. This sentiment score directly influences whether the agent buys, holds or avoids a position.
🛑
Stop-Loss & Take-Profit
The agent automatically sells a position if it falls 5% below the purchase price (stop-loss) or rises 15% above it (take-profit). This protects capital and locks in gains without any manual intervention.
📊
Backtesting
Before trading live, the strategy is validated against one year of real historical price data. The backtest measures total return, win rate, and how many stop-losses and take-profits were triggered — proving the strategy works before risking capital.
📈
S&P 500 Comparison
Every session the agent checks how the broader market is performing by fetching the S&P 500's daily movement. This gives Claude market-wide context — if the whole market is down, the agent is more cautious about new buys.
💾
Persistent Memory
The agent remembers every trade, position and portfolio value across sessions. Cash, holdings and trade history are saved to a local JSON file so the portfolio survives restarts — building a real track record over time.
📓
Obsidian Second Brain
Every trade decision, news summary, and daily report is automatically written as a Markdown note into an Obsidian vault. Over time this builds a searchable knowledge base of the agent's entire decision history and reasoning.
⚙️
Autonomous Scheduling
The agent runs automatically twice a day Monday to Friday using a built-in scheduler. It is deployed inside a NemoClaw sandbox for security and isolation, meaning it operates continuously without any manual input required.
